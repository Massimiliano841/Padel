28 marzo 2026

Autenticazione — sistema admin

Rimossa la registrazione pubblica. L'utente admin è il master e al primo accesso imposta la propria password.

Tab Utenti visibile solo all'admin: crea account, cambia password, elimina utenti.

Log attività

Nuova tab Log con storico immutabile delle modifiche per utente. Traccia: giocatori, tornei, stagioni, risultati partite, accessi. Filtrabile per utente. Sincronizzato in tempo reale su tutti i dispositivi.

Giri del torneo

In creazione torneo è ora possibile scegliere il numero di giri (ripetizioni del tabellone). Le partite vengono generate moltiplicando il tabellone base × giri, con separatori visivi nella lista partite.

Partite extra

Nella tab Partite è possibile aggiungere partite extra a un torneo aperto, scegliendo liberamente i 4 giocatori. Supporto per etichetta opzionale (es. Finale, Spareggio) visibile nella card sotto il punteggio. Badge EXTRA per distinguerle dalle partite generate.

Interfaccia mobile

Layout adattivo per smartphone: bottom navigation con le 4 sezioni principali + menu "Altro" per Tornei, Stagioni, Log e Utenti. Header fisso con logo, stato connessione e chip utente. Ogni pagina ottimizzata per schermi piccoli.

Fix e miglioramenti

Risolto bug CSS che causava la visualizzazione permanente del modal di conferma su tutte le pagine.

Risolto errore di sintassi JS (setStatus orfana) che bloccava l'avvio dell'app.

Migrazione da Firebase/JSONBin a Supabase REST API per connettività stabile e scritture affidabili.

Numero versione visibile in fondo all'app.

Rimosso

Registrazione pubblica — gli account si creano solo dall'admin.

Pulsante "Cancella log" — il log è ora immutabile.

Form di configurazione Firebase/JSONBin — non più necessario.
